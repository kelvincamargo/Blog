Arduino

O Arduino é uma plataforma de prototipagem eletrônica de código aberto que permite criar projetos interativos e automatizados. Ele é composto por uma placa com um microcontrolador (um pequeno computador) que pode ser programado para controlar sensores, motores, luzes e outros dispositivos. O Arduino é amplamente utilizado por hobbyistas, educadores e profissionais para desenvolver protótipos de eletrônicos de forma rápida e acessível.

Linguagem de programação do Arduino

A linguagem de programação do Arduino é baseada em C/C++, mas simplificada para facilitar o uso de iniciantes. O código é escrito no Arduino IDE (Ambiente de Desenvolvimento Integrado), que compila e carrega o código diretamente na placa.

Projetos realizados

1. Controlando um Servo Motor

Nesse projeto foi realizado o controle do movimento de um servo motor usando um potenciômetro. O servo motor é um tipo de motor que pode ser posicionado em ângulos específicos. O potenciômetro vai permitir que você ajuste a posição do servo motor manualmente. 


# Exemplo em C++:

#include Servo.h

    Servo meuServo;
    
    const int pinoPot = A0; 

    int leituraPot;

    int angulo;           
    
    void setup() {

      Serial.begin(9600);

      meuServo.attach(9);

    }
    
    void loop() {

      leituraPot = analogRead(pinoPot); 

      Serial.println(leituraPot);      
    
      angulo = map(leituraPot, 0, 1023, 0, 180);

      meuServo.write(angulo);  

      delay(10); 

    }

2. Semáforo com millis

Projeto realizado de um circuito de um semáforo utilizando a função millis():


                unsigned long agora = 0;

                int ledVerde = 11;

                int ledLaranja = 12;

                int ledRed = 13;
                
                void setup() {

                  pinMode(ledRed, OUTPUT);

                  pinMode(ledLaranja, OUTPUT);

                  pinMode(ledVerde, OUTPUT);
                  
                  agora = millis();  

                }
                
                void loop() {

                  unsigned long tempoDecorrido = millis() - agora;

                  Serial.println(tempoDecorrido);
                  
                  if (tempoDecorrido <= 6000) {

                    digitalWrite(ledVerde, HIGH);

                    digitalWrite(ledLaranja, LOW);

                    digitalWrite(ledRed, LOW);

                  } else if (tempoDecorrido <= 7000) {

                    digitalWrite(ledVerde, LOW);

                    digitalWrite(ledLaranja, HIGH);

                    digitalWrite(ledRed, LOW);

                  } else if (tempoDecorrido <= 16000) {

                    digitalWrite(ledVerde, LOW);

                    digitalWrite(ledLaranja, LOW);
                   
                    digitalWrite(ledRed, HIGH);

                  } 
                  
                     else{
                   
                    agora = millis();
                
                        }

                }

Conclusão

Em conclusão, o Arduino é uma plataforma poderosa e acessível para quem deseja aprender e aplicar eletrônica e programação de maneira prática. Sua simplicidade, aliada a uma comunidade ativa e recursos abundantes, torna a criação de projetos interativos mais fácil e divertida. A linguagem de programação, baseada em C/C++, oferece um ponto de partida ideal para iniciantes, permitindo que qualquer pessoa possa começar a desenvolver soluções criativas. Seja para hobby ou para fins educacionais e profissionais, o Arduino proporciona uma excelente oportunidade de explorar o mundo da tecnologia e da inovação.
